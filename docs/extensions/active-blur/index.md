# Active Blur

Плагин обоев для KDE Plasma, добавляющий регулируемое размытие фонового изображения, когда активно любое окно. Сохраняет поведение стандартного плагина «Изображение», добавляя настройки радиуса размытия, задержки анимации и затемнения.

![active-blur-1](./img/active-blur-1.gif)

## Особенности

- Размытие и опциональное затемнение при активном окне
- Слайд‑шоу режим и предпросмотр живой миниатюры
- Тонкая настройка радиуса, задержки анимации и цвета затемнения
