## Установка заставки

1. Откройте «Параметры системы»
2. Перейдите в «Цвета и оформления» → «Заставка»
3. Нажмите «Загрузить...»
4. В поиске введите «**{{ $frontmatter?.appstream?.name }}**»
5. Нажмите «Установить», выберите заставку и примените изменения
