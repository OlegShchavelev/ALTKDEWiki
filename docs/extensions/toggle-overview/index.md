# Toggle Overview

Минималистичный виджет-кнопка для быстрого открытия эффекта «Обзор» без использования горячих углов или сочетаний клавиш. Подходит для тех, кто предпочитает клик по панели вместо жестов или хоткеев.

![toggle-overview-1](./img/toggle-overview-1.gif)

## Особенности

- Открывает режим «Обзор» по одному клику
- Удобное размещение на панели или рабочем столе
- Минимальные настройки внешнего вида
- Работает поверх стандартных настроек эффектов

<!--@include: @extensions/.parts/show-install-steps.md-->

## Не работает кнопка?

::: warning Проверьте эффект «Обзор»
Если по нажатию ничего не происходит, откройте «Параметры системы» → «Диспетчер окон» → «Эффекты рабочего стола» и убедитесь, что эффект «Обзор» включён. Затем повторно нажмите на кнопку виджета.
:::
