# Shader Wallpaper

Плагин обоев для KDE Plasma, позволяющий использовать интерактивные шейдеры в качестве фона рабочего стола. Доступно более 80 пресетов разной стилистики, а также гибкие настройки скорости, цвета и других параметров шейдеров.

<GalleryALT />

## Особенности

- Более 80 встроенных шейдеров и удобный выбор через меню
- Настройка скорости, цвета и параметров эффекта
- Переключение шейдеров стрелками или колёсиком мыши при наведении
- Возможность установки плагина для экрана блокировки

## Важно

::: danger Внимание: вспышки и мерцания
Некоторые шейдеры содержат яркие вспышки и быстрые смены цветов. Людям, чувствительным к таким эффектам, рекомендуется осторожность при использовании.
:::
