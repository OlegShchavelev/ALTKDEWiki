# Panel Colorizer

Полнофункциональный виджет для глубокой кастомизации панелей с возможностями, вдохновлёнными Latte Dock и оконными менеджерами. Предоставляет исчерпывающий контроль над внешним видом панелей, включая цвета, прозрачность, отступы и поведение отдельных виджетов.

Виджет работает путём инъекции пользовательских стилей в панель, позволяя добиться внешнего вида статус-бара без модификации самих виджетов или замены стандартной панели Plasma.

## Основные возможности

### Настройка фона панели

- Скрытие нативного фона панели (полная прозрачность)
- Регулировка прозрачности стандартного фона
- Отключение тени панели
- Симуляция постоянно плавающей панели
- Принудительный плавающий режим даже при развёрнутых окнах
- Пользовательский фон с градиентами и изображениями
- Настройка радиуса закругления углов фона

### Унифицированные фоны виджетов

- Объединение нескольких виджетов под единым фоном
- Индивидуальная настройка отступов для каждого виджета
- Контроль высоты и выравнивания элементов
- Линейные индикаторы для визуального выделения виджетов
- Поддержка различных форм фона (прямоугольник, округлый)

### Настройка отдельных виджетов

- Принудительное изменение цвета текста и иконок для конкретных виджетов
- Чёрный или белый список виджетов для применения эффектов
- Отключение собственного фона виджетов
- Индивидуальные отступы и промежутки

### Система пресетов

- Создание и сохранение неограниченного количества пресетов
- Автоматическое переключение на основе условий:
  - Наличие развёрнутых или полноэкранных окон
  - Присутствие активных окон на рабочем столе
  - Цветовая схема системы (светлая/тёмная)
  - Пользовательские скрипты с любыми условиями
- Превью пресетов для быстрого визуального сравнения

### Расширенные возможности

- Управление через D-Bus для автоматизации и скриптов
- Скрытие/показ панелей программно
- Применение настроек ко всем панелям одновременно
- Коррекция контраста для всех цветовых режимов
- Настройка толщины панели

::: warning Модификация системы
Виджет модифицирует код панелей на низком уровне и может потенциально привести к проблемам в работе. Рекомендуется создать резервную копию конфигурации Plasma перед установкой.
:::

::: info Настройка
После установки виджет необходимо добавить на панель, которую требуется настроить. Все параметры кастомизации доступны в настройках виджета, где можно создавать пресеты и настраивать автоматическое переключение.
:::

## Примеры использования

Типичные сценарии применения виджета:

- Создание прозрачной панели в стиле macOS с размытым фоном
- Имитация статус-баров оконных менеджеров (i3, bspwm, awesome)
- Группировка виджетов в островки с округлым фоном
- Автоматическое изменение прозрачности панели при развёрнутых окнах
- Создание минималистичных панелей с индивидуальными фонами для каждого виджета
