# KDE Modern Clock

Современный виджет часов с минималистичным дизайном, вдохновлённый популярным скином Mond для Rainmeter. Написан полностью на QML и JavaScript, делая акцент на эстетике и чистоте визуального представления.

Виджет отличается необычным подходом к отображению информации, выделяя день недели крупным шрифтом и располагая остальные элементы вокруг него. Идеально подходит для размещения на рабочем столе в качестве визуального акцента.

![kde-modern-clock-1](./img/kde-modern-clock-1.jpg)

## Основные возможности

### Настройка отображения

- Выбор между 12-часовым и 24-часовым форматом времени
- Настройка размера шрифта для всех элементов
- Изменение цвета текста для адаптации под обои
- Регулировка расположения элементов на экране

### Информационные блоки

- Крупное отображение дня недели как центральный элемент
- Текущая дата с месяцем
- Время в выбранном формате
- Минималистичная компоновка без лишних элементов

## Дизайн-концепция

Виджет следует философии "меньше — значит больше", используя типографику как основное средство выразительности. В отличие от традиционных часов, где главный акцент на времени, Modern Clock выделяет день недели, создавая уникальную визуальную иерархию.

::: tip Для любителей минимализма
Виджет лучше всего смотрится на чистых обоях с достаточным пространством вокруг. Рекомендуется размещать его в центре или по краям рабочего стола для максимального эстетического эффекта.
:::
