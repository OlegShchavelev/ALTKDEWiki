# Базовые сценарии работы с файловым менеджером Dolphin в KDE Plasma

## Введение

**Dolphin** — это стандартный файловый менеджер среды рабочего стола **KDE Plasma**. Он предназначен для повседневной работы с файлами и каталогами и сочетает в себе простоту для начинающих пользователей и широкий набор возможностей для продвинутых сценариев.

Если вы ранее использовали Windows Explorer, Finder (macOS) или файловые менеджеры других Linux-окружений (например, Nautilus в GNOME), Dolphin будет интуитивно понятен, но при этом предложит ряд дополнительных функций, характерных именно для KDE.

## Запуск Dolphin

Запустить Dolphin можно несколькими способами:

* **Через меню приложений KDE**
  Меню → Система → Dolphin (или «Файловый менеджер»)

  ![Dolphin в Меню запуска приложений](./img/dolphin-1.png)

* **Через значок на панели** (обычно присутствует по умолчанию)

:::tip
Если значок не отображается, его можно добавить через контекстное меню в Меню запуска приложений:  
**Правый клик по пункту меню Dolphin → Закрепить на панели задач**.

![Добавление значка Dolphin на панель задач](./img/dolphin-2.png)

:::
* **С помощью сочетания клавиш**
  `Meta (Win) + E`
* **Из терминала**

  ```bash
  dolphin
  ```

После запуска откроется главное окно файлового менеджера.

![Главное окно Dolphin](./img/dolphin-3.png)

## Обзор интерфейса Dolphin

### Основные элементы окна

1. **Панель меню** (по умолчанию убрана в кнопку «бургер-меню»)

:::tip
Для отображения меню в виде панели нужно нажать на кнопку меню («бургер-меню») и в открывшемся списке выбрать `Дополнительно` и `Показать меню со всеми действиями` либо же просто управлять режимом отображения панели меню через комбинацию клавиш `Ctrl + M`.

![Показ панели меню Dolphin](./img/dolphin-11.png)
:::

2. **Панель навигации**

   * Назад / Вперёд
   * Вверх (к родительскому каталогу)

![Панель навигации](./img/dolphin-4.png)

3. **Адресная строка**

   * Отображает текущий путь
   * Может работать в двух режимах:

     * *Навигационный* (кнопки каталогов)

     ![Адресная строка в навигационном режиме](./img/dolphin-5.png)

     * *Текстовый* (полный путь, переключение по `Ctrl + L`)

     ![Адресная строка в текстовом режиме](./img/dolphin-6.png)

4. **Панель инструментов**
   
   По обе стороны от адресной строки расположены важные инструменты управления:
   * **Настройка режимов просмотра**
   * **Две панели** (`F3`) — включает двухпанельный режим.
   * **Поиск** (`Ctrl + F`) — открывает строку поиска. 
   * **Кнопка меню** («бургер-меню») — доступ к полному списку настроек и функций. Панель меню по умолчанию скрыта.

   ![Кнопки управления Dolphin](./img/dolphin-10.png)

  :::tip
  **Добавление и удаление кнопок панели инструментов**
  На панели инструментов файлового менеджера часто используемые функции расположены в виде кнопок.
  Можно добавлять и удалять кнопки, располагать их в нужном порядке.

  Для этого нужно:

  1. Открыть пункт меню **Настройка** (в панели меню, либо через кнопку «бургер-меню»)
  2. Перейти в раздел **Настроить панели инструментов**

  * В левой колонке есть все возможные команды и разделители, в правой - те ярлыки, которые есть сейчас на панели. Для быстрого поиска над панелями есть фильтр для текстового поиска.
  * Стрелочки в центре предназначены для перемещения ярлыков.
  * Для добавления нового ярлыка найдите нужную команду, нажмите стрелочку «вправо» или дважды щелкните по ней мышью. Ярлык будет скопирован в правую колонку. Таким же образом можно вставить разделитель.
  * Для удаления ярлыка найдите его в правой колонке и нажмите стрелочку «влево» или дважды щелкните по нему мышью.

  ![Добавление и удаление кнопок панели инструментов](./img/dolphin-33.png)
  :::

5. **Боковая панель (Places / Места)**
   Содержит быстрый доступ к:

   * Домашнему каталогу и его подкаталогам (Документы, Загрузки, Изображения и т.д.)
   * Корзине
   * Сетевым ресурсам
   * Недавно измененным файлам и расположениям.
   * Подключённым дискам и флеш-накопителям

   ![Боковая панель Dolphin](./img/dolphin-7.png)

6. **Основная область**
   Здесь отображается содержимое текущей папки.

   ![Основная область Dolphin](./img/dolphin-8.png)

7. **Строка состояния**
   Показывает количество файлов, размер выделенных элементов и другую информацию.

   ![Строка состояния Dolphin](./img/dolphin-9.png)

## Навигация по файловой системе

### Переход между папками

* **Двойной щелчок мыши** — открыть папку
* **Одинарный щелчок** (если включён соответствующий режим) — открыть папку

 ![Настройка режима щелчка мышью](./img/dolphin-12.png)

* **Кнопка «Назад» / «Вперёд»**
* **Кнопка «Вверх»** — переход на уровень выше
* **Использование боковой панели** — быстрый доступ к важным каталогам

![Кнопки навигации](./img/dolphin-14.png)

### Работа с адресной строкой

* Нажмите `Ctrl + L`, чтобы вручную ввести путь (например: `/home/user/Documents`)
* Нажмите `Esc`, чтобы вернуться в навигационный режим

![Адресная строка](./img/dolphin-13.png)

* **Значки** — удобен для изображений и мультимедиа

* Через кнопки на панели инструментов

Сортировка доступна:

* Через заголовки столбцов (в табличном виде)

![Сортировка](./img/dolphin-19.png)

* Через меню **Вид → Сортировка**

## Основные операции с файлами

### Выделение файлов

* Один файл — одинарный щелчок
* Несколько файлов:

  * `Ctrl + щелчок` — выбор отдельных элементов
  * `Shift + щелчок` — диапазон
* Выделить всё: `Ctrl + A`

### Копирование, перемещение, удаление

* **Копировать**:
  `Ctrl + C` → `Ctrl + V`
* **Вырезать (переместить)**:
  `Ctrl + X` → `Ctrl + V`
* **Удалить в корзину**:
  `Delete`
* **Удалить без корзины**:
  `Shift + Delete`

Также поддерживается **перетаскивание мышью (Drag & Drop)**:

* Перетащить с зажатой левой кнопкой
* При отпускании можно выбрать: копировать, переместить или создать ссылку

![Перетаскивание мышью](./img/dolphin-20.png)

## Работа с корзиной

Удалённые файлы по умолчанию попадают в **Корзину**.

В корзине можно:

* Восстановить файл
* Очистить корзину полностью

![Корзина](./img/dolphin-21.png)

Корзина доступна:

* Через боковую панель
* По пути: `trash:/`

## Создание файлов и папок

### Создание папки

* Правый клик → **Создать → Папку**
* Или сочетание: `F10`

### Создание файлов

Через контекстное меню:

* **Создать → Текстовый файл**
* **Создать → [шаблоны файлов]**

Шаблоны можно расширять, добавляя файлы в каталог:

```
~/.local/share/templates/
```

Подробнее об этом можно почитать в разделе [Настройка меню «Создать» в Dolphin](../dolphin-templates/index.md).

### Отображение точного времени создания/изменения файлов

По умолчанию в Dolphin дата создания/изменения файлов отображается относительно текущей даты: минуту назад, час назад, день назад и т.д. 

Чтобы отобразить точное время создания/изменения файлов, нужно:

1. Открыть пункт меню **Настройка** (в панели меню, либо через кнопку «бургер-меню»)
2. Перейти в раздел **Настроить Dolphin**
3. В боковой панели выбрать **Вид**
4. Выбрать вкладку **Содержимое**
5. В разделе **Формат дат** отметить опцию **Абсолютный (...)**


![Пункт меню Настройка Dolphin](./img/dolphin-32.png)
![Абсолютный формат даты создания/изменения файлов](./img/dolphin-31.png)

## Контекстное меню (правый клик)

Контекстное меню — ключевой элемент работы в Dolphin. Оно меняется в зависимости от выбранного объекта и может содержать:

* Открыть / Открыть с помощью
* Копировать, Вырезать, Переименовать
* Свойства
* Архивировать / Распаковать
* Действия от установленных приложений (например, Git, редакторы)

## Работа с вкладками и разделением окна

### Вкладки

Dolphin поддерживает вкладки, как в браузере:

* Новая вкладка: `Ctrl + T`
* Закрыть вкладку: `Ctrl + W`
* Переключение: `Ctrl + Tab`

![Вкладки](./img/dolphin-22.png)

### Разделение окна

Можно разделить окно на две панели:

* `F3` — включить/выключить разделение
* Удобно для копирования файлов между каталогами

![Разделение окна](./img/dolphin-23.png)

## Поиск файлов

### Быстрый поиск

* Просто начните вводить текст в текущей папке
* Dolphin отфильтрует файлы по имени

### Расширенный поиск

* Используйте строку поиска
* Можно искать:

  * По имени
  * По содержимому (если установлен индексатор, например Baloo)

![Поиск](./img/dolphin-24.png)

## Подключение внешних устройств и сетевых ресурсов

### Внешние носители

Флешки, внешние диски:

* Автоматически отображаются в боковой панели
* Монтируются по клику
* Перед извлечением используйте «Безопасно извлечь»

![Внешние носители](./img/dolphin-25.png)

### Сетевые ресурсы

Dolphin поддерживает:

* SMB/CIFS (Windows-сети)
* FTP, SFTP
* WebDAV

Доступ осуществляется через:

* Боковую панель → Сеть
* Или ввод адреса вида:

  ```
  smb://server/share
  ```

![Сетевые ресурсы](./img/dolphin-26.png)

![Samba](./img/dolphin-27.png)

* Посмотреть размер и тип файла
* Изменить имя
* Настроить права доступа (чтение, запись, выполнение)
* Изменить владельца (при наличии прав)

## Заключение

Dolphin — мощный и гибкий файловый менеджер, который подходит как для повседневных задач, так и для более сложных сценариев. Новым пользователям KDE Plasma рекомендуется:

* Освоить навигацию и контекстное меню
* Использовать вкладки и разделение окна
* Постепенно знакомиться с расширенными возможностями

Даже базового освоения Dolphin достаточно, чтобы эффективно работать с файлами в Linux-системах на базе KDE.
